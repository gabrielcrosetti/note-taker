## Note Taker

This app is meant to be used as a note taker and was made using Express.js, Node.js, and Heroku. The user is able to add notes, store them, and view them later.

## Technologies Used

* HTML
* CSS
* JavaScript
* Express js
* Node js
* Git
* Heroku
